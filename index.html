Dashboard For Temple Harrisburg Training

The Continuning Education Unit at Temple Harrisburg have trouble collecting accumulative information regarding attendance. The task is very time-consuming because the LMS offers no way to automate the process. My Capstone group is working to develop a dashboard that will handle the extraction of data and visualize it to highlight the unit's performance. We will be using Microsoft Power BI tools to help the team improve their proficiency!

School Directory

Using C# and Windows Forms, my Component-Based design group built a system allowing for the registration of students and faculty. The students are classified as undergraduate or graduate and the faculty as professor or chairperson. Using validation controls, information entry is checked for requirement satisfaction. If searching for associate, the ID field is compared to a database of current entries. Each category is held within its own subclass and the connection to the database was also in a separate class, all of which connected to the form.
